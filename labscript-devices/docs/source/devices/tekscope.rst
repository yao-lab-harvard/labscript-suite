Tektronix Oscilloscope
======================

A device for controlling Tektronix oscilloscopes using the standard VISA interface.

.. autosummary::
   labscript_devices.TekScope.labscript_devices
   labscript_devices.TekScope.blacs_tabs

Installation
~~~~~~~~~~~~

This wrapper requires PyVISA and a compatible VISA installation. Free versions are
provided by NI and Keysight (NI preferred if already using NI DAQs).
